# 审计取证单生成工具

将 Excel 审计问题清单按"问题标题"大类分组，自动填入 Word 取证单模板，生成合并后的取证单 Word 文档。

## ✨ 特性

- 🚀 **零依赖** — 提供 EXE 可执行文件，无需安装 Python
- 📊 **智能分组** — 按问题标题自动分组，按定性分小节，中文编号
- 🔄 **去重合并** — 连续相同的违反依据自动合并为"以上事项违反了..."
- 📝 **模板驱动** — 基于 Word 模板生成，保留原格式
- 📄 **多取证单合并** — 多个大类自动分页，合并为一个文档

## 快速开始

### 方式一：EXE 直接运行（免安装 Python）

从 [Releases](https://github.com/liujiasende/audit-evidence-form/releases) 下载最新的 `取证单生成CLI.exe`。

```bash
取证单生成CLI.exe --excel "问题清单.xlsx" --template "取证单模板.docx"
```

### 方式二：Python 运行

需要 Python 3.8+ 及依赖：

```bash
pip install openpyxl python-docx lxml
python 取证单生成CLI.py --excel "问题清单.xlsx" --template "取证单模板.docx"
```

## 使用方法

### 参数说明

| 参数 | 缩写 | 必填 | 说明 |
|------|------|------|------|
| `--excel` | `-e` | 是 | Excel 问题清单文件路径 |
| `--template` | `-t` | 是 | Word 取证单模板路径 |
| `--output` | `-o` | 否 | 输出目录（默认：Excel 同目录下"生成后大取证单"文件夹） |

### Excel 格式要求

Excel 第一行为表头，从第二行开始为数据：

| 列 | 内容 | 说明 |
|----|------|------|
| B（第2列） | 序号 | 必填，空行自动跳过 |
| C（第3列） | 问题标题 | 用于分组（相同标题归为同一取证单） |
| D（第4列） | 定性 | 定性描述 |
| E（第5列） | 审计发现问题描述 | 具体问题详情 |
| F（第6列） | 定性处理依据 | 违反的法规/制度条款 |
| G（第7列） | 责任部门 | 责任归属部门 |

### Word 模板要求

模板中必须包含两个占位符：
- `#1` → 自动替换为问题标题（问题大类名）
- `#2` → 自动替换为该大类下所有问题的详细内容

### 示例

本仓库附带示例文件，可以直接测试：

```bash
取证单生成CLI.exe -e "示例_问题清单.xlsx" -t "示例_取证单模板.docx" -o "输出"
```

## 核心逻辑

1. 读取 Excel 数据（从第2行开始，跳过空行）
2. 按"问题标题"列分组（相同标题归为同一大类）
3. 每个大类内按"定性"分小子组，生成中文编号（（一）（二）...）
4. 对违反依据做去重合并：连续相同依据 → 合并为"以上事项违反了XXX"
5. 填充 Word 模板：第一个大类填原模板，后续大类复制模板并插入分页符
6. 清除空段落，保存合并文件
